# Seva Path Charitable Trust

![Seva Path Logo](img/sevapath-logo.png)

> **"સેવા એ જીવનનો પથ છે" - Seva is the path of life**

Seva Path Charitable Trust is a non-profit organization (NGO) working since 2009 to empower tribal and rural communities in Gujarat, India. Our mission is to bridge the gap in essential services through healthcare, education, skill training, and animal welfare.

## 🌟 Key Initiatives

We focus on holistic community development through several dedicated programs:

-   📚 **Education & Skill Development:** Empowering tribal youth through quality education, digital literacy, and vocational training.
-   💚 **Healthcare & Life Care:** Bringing medical expertise and essential medicines to remote areas through free medical camps and mobile clinics.
-   🐾 **Animal Welfare:** Rescuing, treating, and providing shelter to injured and abandoned animals in tribal regions.
-   🏹 **Johar Haat:** A platform for promoting and selling authentic tribal handicrafts, preserving heritage, and supporting local artisans.
-   🤝 **Youth Programs:** Cultivating leadership and character building among tribal youth through workshops and community service.

## 🚀 Project Overview

This repository contains the official website for Seva Path Charitable Trust. It is built as a highly responsive, performant, and SEO-optimized static website.

### 🛠 Technologies Used

-   **Frontend:** HTML5, CSS3, JavaScript (ES6+)
-   **Styling:** Bootstrap 5, Custom CSS (Vanilla)
-   **Icons:** FontAwesome 6, Simple Line Icons
-   **Plugins:** Owl Carousel, Magnific Popup, GSAP (Animations)
-   **Framework Base:** Porto Admin & Frontend Template

### 📈 SEO & Social Sharing

-   **Open Graph (OG) Tags:** Fully integrated for rich link previews on Facebook, LinkedIn, WhatsApp, and more.
-   **Meta Tags:** Optimized titles, descriptions, and keywords for better search engine visibility.
-   **Accessibility:** Semantic HTML structure and responsive design for all device types.
